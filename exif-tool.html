// Three things you'll feel immediately

A small tool that does exactly what photographers ask of it.

No subscriptions, no upload-to-cloud, no compression. Open a photo, pick your overlay, export. That's it.

01 / Comprehensive
0EXIF fields

Stamp 15 EXIF parameters.

Camera, lens, shutter, aperture, ISO, focal length, subject distance, white balance, flash, exposure compensation, metering mode, exposure program, date & time, copyright & GPS — any combination.

02 / Customisable

Any corner, any colour, any size.

Place the overlay top-left, top-right, bottom-left or bottom-right. Pick white, yellow or black text. Scale from subtle to bold to suit every composition.

03 / Safe by design

Your original is never modified.

Every export is written to a brand-new file with an _exif suffix, saved alongside the source. The photograph you shot stays untouched — pixel for pixel.

Just shipped · Version 1.1

Four new ways to control the stamp.

Version 1.1 builds on everything from the first release — with more control over how the overlay reads, a place for your own words, and where the finished files land.

01 Layout

One single line, or all four corners.

Keep it minimal with a single strip of data along the top or bottom edge — or spread the full detail across multiple lines in any of the four corners. Your call, shot by shot.

02 Field labels

Show the labels, or let the values speak.

Display each field with its name — Shutter: 1/320s — or strip the labels for a clean, compact read of just the numbers. A single toggle switches the whole overlay.

Shutter: 1/320s 1/320s
03 Notes

Add a note in your words.

A new notes field lets you stamp a short caption — a location, a subject, a title — right alongside the data. Up to two lines, sitting neatly with the rest of the overlay.

“Kingfisher at dawn — Bharatpur”
04 Output

Send finished files to a folder you choose.

Pick a destination folder and every stamped image is saved there — keeping your exports tidy and separate from the originals, which still stay exactly as you shot them.

// Safety & trust

A small EXE, built with care.

Because this is a Windows executable from a one-person studio, your browser or Windows may pause for a moment the first time you open it. Here is exactly what's inside the file, why those warnings appear, and how to proceed safely.

A · What's in the file

Nothing but the tool itself.

EXIF Overlay Tool is a small standalone application. It does one job — read EXIF data, stamp it onto a copy, save the copy. Nothing more.

  • No background services, no scheduled tasks
  • No telemetry, no analytics, no “phone home”
  • No internet access required to run
  • No registry changes outside its own preferences
  • Uninstall by deleting the file — that's it
B · How you can verify

Don't take my word — check it yourself.

Three independent ways to confirm the file you downloaded is the file I published, and that it is clean.

  • SHA-256 hash published next to the download — match it against the file you saved
  • VirusTotal report linked on the download page (60+ AV engines scan it for you)
  • Block your firewall — the tool will continue to work; it never needs the network

About the Windows warning you may see

Windows shows a blue SmartScreen dialog the first time it sees a new EXE from a small publisher. This is reputation-based — not a virus alert. A code-signing certificate is on the roadmap, but until then, here's the safe walk-through:
